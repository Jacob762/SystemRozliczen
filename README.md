# System Rozliczeń

## Wymagania

- Node 20
- PNPM (preferowany sposób instalacji - Volta - uwaga, wymaga dodatkowej zmiennej środowiskowej)
- Java

## Instalacja

```bash
git clone https://github.com/Jacob762/SystemRozliczen.git
cd SystemRozliczen
pnpm install
turbo dev
```
